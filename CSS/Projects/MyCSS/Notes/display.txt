-> The default display value for most elements is block or inline.


display: none  -> hide element also don't leave blank space behind.
display: block  -> start elem on a new line and takes as mush space it can
display: inline  -> does not start with on a new line and takes only required space.

visibility: hidden  -> hide element like display: none but leave a blank space behind it

block elements: 
-> h1 - h6
-> p
-> div
-> li

inline elements: 
-> span
-> img
-> a
