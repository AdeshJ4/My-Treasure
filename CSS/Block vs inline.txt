In CSS, HTML elements are generally classified as either block-level elements or inline elements. 



1. Block-Level Elements: 

Layout: Block-level elements start on a new line and take up the full width available by default.
Width and Height: You can set width and height properties.
Margin and Padding: Margins and padding can be set on all four sides (top, right, bottom, left).

Characteristics: 
display: block;

Common Block-Level Elements:
<div>
<p>
<h1>, <h2>, <h3>, etc.
<ul>, <ol>, <li>
<form>
<header>, <footer>, <section>, <article>

<div style="background-color: lightblue;">
  This is a block-level element.
</div>
<p style="background-color: lightgreen;">
  This is another block-level element.
</p>



2. Inline Elements

Characteristics:
Display Property: display: inline;
Layout: Inline elements do not start on a new line. They occupy only the necessary width required by their content and can sit next to other inline elements.
Width and Height: Width and height properties do not apply.
Margin and Padding: Margins and padding apply only to the left and right sides. Top and bottom margins and padding do not affect the layout.

Common Inline Elements:
<span>
<a>
<img>
<strong>, <em>
<br>
<label>
<input>, <button>

Example:
<span style="background-color: lightcoral;">
  This is an inline element.
</span>
<a href="#" style="background-color: lightgoldenrodyellow;">
  This is another inline element.
</a>


3. Inline-Block Elements

There is also a hybrid display type called inline-block that combines characteristics of both block and inline elements.

Characteristics:
Display Property: display: inline-block;
Layout: Inline-block elements are placed inline (like inline elements), but you can set width and height (like block elements).
Width and Height: You can set width and height properties.
Margin and Padding: Margins and padding can be set on all four sides.

example: 
<div style="display: inline-block; width: 200px; height: 100px; background-color: lightpink;">
  This is an inline-block element.
</div>
