Tailwind CSS provides utility classes to use **CSS Grid**, which is a powerful 2D layout system to arrange items in **rows and columns**.

## ğŸ“¦ Basic Usage

```html
<div class="grid grid-cols-3 gap-4">
  <div class="bg-red-300 p-4">1</div>
  <div class="bg-blue-300 p-4">2</div>
  <div class="bg-green-300 p-4">3</div>
  <div class="bg-yellow-300 p-4">4</div>
</div>
```

### ğŸ” Explanation:
- `grid`: turns the container into a grid.
- `grid-cols-3`: creates **3 equal columns**.
- `gap-4`: adds space between the grid items.

---

## ğŸ“ Grid with Rows and Columns

```html
<div class="grid grid-cols-2 grid-rows-2 gap-2">
  <div class="bg-pink-300 p-4">A</div>
  <div class="bg-purple-300 p-4">B</div>
  <div class="bg-teal-300 p-4">C</div>
  <div class="bg-orange-300 p-4">D</div>
</div>
```

- `grid-rows-2`: creates **2 rows**
- `grid-cols-2`: creates **2 columns**
- You'll see a 2x2 layout.

## ğŸ“ Responsive Grid

```html
<div class="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4">
  <!-- Grid items -->
</div>
```

- Adjusts column count based on screen size:
  - 1 column on mobile
  - 2 on small screens (â‰¥640px)
  - 3 on medium (â‰¥768px)
  - 4 on large (â‰¥1024px)

## ğŸ¯ Span Columns or Rows

```html
<div class="grid grid-cols-3 gap-4">
  <div class="col-span-2 bg-lime-300 p-4">Takes 2 columns</div>
  <div class="bg-gray-300 p-4">Takes 1 column</div>
</div>
```

- `col-span-2`: makes the item span 2 columns.
