Join: 
-> A JOIN clause is used to combine rows from two or more tables, based on a related column between them.


Inner Join: 

-> The INNER JOIN selects records that have matching values in both tables that match the specified condition.
-> MySQL assumes it as a default Join, so it is optional to use the Inner Join keyword with the query.








CREATE DATABASE OK;
SHOW DATABASES;
USE OK;
SHOW TABLES;
CREATE TABLE STUDENT (
    STUDENT_ID INT PRIMARY KEY,
    STUDENT_NAME VARCHAR(50),
    CITY VARCHAR(50),
    TECH_ID INT
);

CREATE TABLE TECHNOLOGIES (
    TECH_ID INT PRIMARY KEY AUTO_INCREMENT,
    TECH_NAME VARCHAR(50)
);
